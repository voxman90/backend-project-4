# page-loader
Page loader utility

## Usage

```shell
$ page-loader -h
Usage: page-loader [options] <url>

Page loader utility

Options:
  -V, --version      output the version number
  -o --output [dir]  output dir (default: "/home/user/current-dir")
  -h, --help         display help for command
```
